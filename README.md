## What is Taskion?

**Taskion** is a modern, fast, and intuitive project management tool for teams of all sizes.  
It is fully self-hosted, giving you complete control over your data and deployment.

### Key Features
- Kanban board with drag & drop
- Gantt chart for project timelines
- Task management with priorities, deadlines, and attachments
- Comments and notifications for team collaboration
- User roles and permissions
- Multi-language support (Czech, English, German, Spanish)
- Dark & light mode
- Personal notes and reminders
- Dashboard overview
- Secure authentication (JWT, hashed passwords)
- Free updates
